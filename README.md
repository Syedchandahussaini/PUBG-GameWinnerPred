# 🎮 PUBG Game Winner Prediction

## 📌 Project Overview
This project focuses on predicting PUBG match winning probability (`winPlacePerc`) using Machine Learning regression models and gameplay statistics.

The project includes:
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Multiple Machine Learning Models
- Model Comparison
- Feature Importance Analysis

---

## 🎯 Problem Statement

### Task 1
Prepare a complete data analysis report on PUBG gameplay data.

### Task 2
Build predictive Machine Learning models to:
- Predict PUBG win probability
- Identify important gameplay factors affecting performance

---

## 🕹️ Domain
Gaming Analytics / Machine Learning

---

## 📂 Dataset Information

The dataset contains PUBG player gameplay statistics including:
- kills
- damageDealt
- boosts
- walkDistance
- heals
- matchDuration
- winPlacePerc

Target Variable:
```python
winPlacePerc
```
